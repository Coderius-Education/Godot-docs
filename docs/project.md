# Nieuw project aanmaken

In deze stap maak je een nieuw project aan in Godot via het Project Manager venster.

## Stappen

1. Open Godot — je ziet het **Project Manager** venster
2. Klik op **New Project**
3. Geef je project een naam (bijv. `mijn-eerste-game`)
4. Klik op **Browse** en kies een map op je computer (bijv. `Documenten`)
5. Klik op **Create & Edit** — de editor opent nu

## Er gaat iets mis

<details>
<summary>Ik zie de knop "New Project" niet</summary>

**Oorzaak:** Je kijkt mogelijk naar een ander scherm in de Project Manager.

**Oplossing:**
- De knop **New Project** staat rechtsboven in het Project Manager venster
- Als je een leeg scherm ziet zonder knoppen: sluit Godot en open het opnieuw via het `.exe`-bestand

</details>

<details>
<summary>De map die ik kies geeft een foutmelding</summary>

**Oorzaak:** Godot kan geen project aanmaken in een map die al bestanden bevat, of je hebt geen schrijfrechten voor die map.

**Oplossing:**
- Kies **Create Folder** om een nieuwe, lege map aan te maken
- Vermijd mappen zoals `Downloads` of het bureaublad — gebruik `Documenten`

</details>
